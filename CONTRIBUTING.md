# Contributing to llm-armory-website

This repository is the **marketing one-pager** for [`llm-armory`](https://github.com/muslewski/llm-armory).

## Where to file what

| Topic | Repo |
|---|---|
| Tool bugs, CLI features, install failures | **[llm-armory](https://github.com/muslewski/llm-armory/issues/new/choose)** |
| Questions & ideas about the tool | **[llm-armory Discussions](https://github.com/muslewski/llm-armory/discussions)** |
| Website copy / layout / performance | Issues on **this** repo |

## Dev (website)

```bash
npm install
npm run dev      # Vite
npm run build
```

Please keep product install commands accurate (`npm install -g llm-armory` / `npx …`).
